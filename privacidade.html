1. Quem trata os teus dados

A Adaptafit é uma aplicação de acompanhamento nutricional desenvolvida de forma independente. O responsável pelo tratamento dos dados recolhidos através da app és tu — o utilizador é dono dos seus próprios dados; nós apenas os processamos para fazer a app funcionar, conforme descrito abaixo.

2. Que dados recolhemos

Só recolhemos o que é necessário para a app funcionar. Nada disto é vendido a terceiros.

2.1 Dados de conta

DadoPorquê
Email e passwordCriar e proteger a tua conta (login)
NomePersonalizar a app com o teu nome

2.2 Dados de perfil e saúde

DadoPorquê
Idade, género, altura, pesoCalcular as tuas necessidades calóricas (fórmula TDEE)
Nível de atividade e objetivoAjustar a meta calórica diária ao teu objetivo
Peso ao longo do tempoMostrar a tua evolução e ajustar a meta automaticamente
Passos diáriosLidos do Health Connect (Android), só leitura — nunca escrevemos nem alteramos esse dado no teu telemóvel
Copos de águaAcompanhar a tua hidratação diária

2.3 Dados de alimentação

DadoPorquê
Alimentos e refeições registadosCalcular calorias e macronutrientes consumidos
Preferências e restrições alimentaresFiltrar receitas e resultados de pesquisa (ex: vegetariano, sem glúten)
Fotografias de pratos (opcional)Só se usares a função "Foto do prato" — a imagem é enviada para análise e não fica guardada permanentemente nos nossos servidores

Código de grupo e progresso partilhadoSó se decidires criar ou entrar num grupo familiar — os membros do grupo veem o teu progresso de passos no ranking

2.5 Dados técnicos

Informação básica de diagnóstico (ex: se a app teve algum erro), para nos ajudar a corrigir problemas. Não usamos isto para te identificar pessoalmente.

3. Para que usamos os dados

  • Calcular e ajustar a tua meta calórica e de macronutrientes
  • Mostrar o teu progresso (peso, calorias, passos, água)
  • Gerar sugestões de receitas e listas de compras adaptadas a ti
  • Atribuir medalhas e acompanhar desafios (gamificação)
  • Permitir a participação em grupos familiares (só se optares por isso)
  • Mostrar anúncios, apenas na versão gratuita (ver secção 4)
  • Melhorar a app e corrigir problemas técnicos

4. Com quem partilhamos dados

Não vendemos os teus dados a ninguém. Usamos os seguintes serviços de terceiros para a app funcionar:

Firebase (Google) — usado para autenticação (login) e para guardar os teus dados na nuvem (base de dados). Política de privacidade da Google.

Google AdMob — mostra anúncios apenas a utilizadores da versão gratuita (sem subscrição Premium ou teste grátis ativo). Pode recolher identificadores de publicidade para personalizar anúncios, conforme as definições de privacidade do teu Android.

Health Connect (Google) — usado apenas para ler o número de passos diários. A app nunca escreve, altera nem partilha esse dado com mais ninguém.

Google Gemini API — só é usada se ativares as funcionalidades de IA (escrever refeição por texto ou foto), e apenas com a tua própria chave de API pessoal, gerida por ti.

5. Como protegemos os teus dados

  • Toda a comunicação entre a app e os servidores é feita por ligação encriptada (HTTPS) — a app não permite tráfego não encriptado.
  • Os dados são guardados na infraestrutura do Google Cloud (Firebase), com os padrões de segurança da Google.
  • O acesso à tua conta é protegido por password (ou pelo método de login que escolheres).

6. Quanto tempo guardamos os dados

Guardamos os teus dados enquanto a tua conta existir. Se pedires a eliminação da conta, os teus dados pessoais são apagados dos nossos sistemas, exceto quando formos legalmente obrigados a conservar alguma informação por mais tempo.

7. Os teus direitos

De acordo com o Regulamento Geral de Proteção de Dados (RGPD), tens direito a:

  • Aceder aos dados que temos sobre ti
  • Corrigir dados incorretos (podes fazer isto diretamente no Perfil da app, na maioria dos casos)
  • Eliminar a tua conta e os dados associados
  • Exportar os teus dados num formato legível
  • Opor-te a determinados tratamentos de dados (ex: publicidade personalizada)

Para exerceres qualquer um destes direitos, contacta-nos através do email na secção 10.

8. Crianças

A Adaptafit não se destina a crianças menores de 16 anos, e não recolhemos intencionalmente dados de crianças. Se descobrirmos que recolhemos dados de uma criança sem consentimento parental válido, eliminamo-los assim que possível.

9. Alterações a esta política

Podemos atualizar esta política ocasionalmente, por exemplo quando adicionamos novas funcionalidades. A data no topo desta página indica a versão mais recente. Alterações significativas serão comunicadas dentro da app.
